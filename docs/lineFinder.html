(static) terminal4DPrint(board) → {string}

Source:
  • common/lineFinder.js, line 38

Uses procedural code to print out a 4 dimensional board in the terminal in an easy-to read format for debugging or playing the game in the terminal. Thought there was no point in making this function pure.

Parameters:
Name Type Description
board matrix

The 4-dimensional board to print out
Returns:

The text that represents the board we want to display. Can be printed out in the terminal or read to a website for easy implementatoin, etc.
