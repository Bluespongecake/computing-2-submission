tacToe.js is a module to model N-dimensional tic-tac-toe.
Made using Functional Programming techniques - Note that all functions in this module are pure

(static) isFullDraw(board)

Source:
  • tacToe.js, line 96

Checks if the board is in a draw state, i.e is full. This seems like an almost impossible state to reach without any winning conditions present, but I should probably have a function to check for it just in case. For efficiency, this does not call the intensive checkWin() function, even though it was be the first thing I thought of when writing the description. It is designed to be called after checkWin() to see if a draw has occured.

Parameters:
Name Type Description
board matrix

N-dimensional board to check
Returns:

True if the board is in a draw state, false if it is not
